About SXMac

Overview

SXMac is a lightweight, note-like app that can be used as a rich text editor, Markdown editor and preview, and code editor with more than 100 supported languages. If you need specific files or anything else in one place.

Simplicity

While being about simplicity, SXMac combines all needs for editing any kind of text. We don’t always want to set it up, but work out of the box! That is all about it!

Note Taking

SXMac can be used as a note-taking app with a simple and clear interface that gets the job done. The main focus is to preview .md files and dynamically see changes in your editor, with file export, easy import, and a centralized experience.

Unity

SXMac lets you quickly view or edit files. You can easily drop them into the documents folder for recognition or use a file picker box.

Source Files Editor

Code highlighting without effort for any files you need. Yes, you can use it for basic coding with all editor functionality!

Markdown Editor

Syntax highlighting for a clean, clear view.

Preview Rich Markdown

Instantly preview your Markdown file (.md) with all features supported!

Quick Preview Changes

Quickly change the background color and see how it looks on different backgrounds.

Handy Features

Quick actions in the right place! Double-click and perform needed actions fast from a small menu!

Drag and Drop

This makes the app 100x more usable! Select a file from any mess and add it straight to the main menu!

Additional Features

Includes an HTML editor with website preview — write and view HTML pages right in the app! Plus, many more features not mentioned here. Download SXMac to share your opinion and help us improve!

Compatibility

System Requirements

SXMac runs smoothly on macOS 14.6 and later. Whether you're rocking an Intel-based Mac or the latest Apple Silicon (M1, M2, and beyond), we've got you covered!

OS: macOS 14.6 or newer
CPU: Intel x86_64 or Apple Silicon (ARM64)
Storage: 20 MB free space
